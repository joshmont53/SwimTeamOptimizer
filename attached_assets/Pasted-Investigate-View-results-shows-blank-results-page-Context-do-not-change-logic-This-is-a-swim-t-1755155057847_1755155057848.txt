Symptom: When entering via a linear flow (same session), results render; on fresh navigation (â€œView resultsâ€), state is empty so renders nothing.

Fix (front-end): On ResultsPage mount, if results absent/empty, fetch by teamId and hydrate state. Example:

  }
  load();
  return () => { cancelled = true; };
